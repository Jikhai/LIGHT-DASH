Installation de l’environnement de développement Django 2.1.5 sur Windows
Installation de Python 3 64bit

Nous utiliserons l’installeur officiel Python 3.7.2 disponible ici python-3.7.2-amd64.exe

Procédez à l’installation en respectant les étapes suivante

Cochez « Add Python 3.7 to PATH » et cliquez sur « Customize installation »

Cliquez sur Next

Cochez « Install for all users » puis sur « Install »

Installation de Django

Maintenant nous allons procéder à l’installation de Django en utilisant « pip » qui est inclus dans Python 3. Cette installation requiert une connexion internet.

Ouvrez un cmd avec les privilèges administrateurs, vous pouvez le faire en cherchant cmd dans le Menu Démarrer et en faisant un clic droit, « Exécuter en tant qu’administrateur ».

Mettez à jour l’utilitaire pip

py -m pip install --upgrade pip

Installez Django

py -m pip install django==2.1.5

Installez Pillow

py -m pip install pillow==5.1.0
Création de votre projet Django de test

Dans le répertoire de votre choix dans un cmd, exécutez la commande pour créer le projet

django-admin startproject nomduprojet

Maintenant, déplacez-vous dans le répertoire de votre projet

cd nomduprojet

Et enfin, lancer le serveur web de développement

python manage.py runserver

Le serveur est accessible en local via l’url suivante http://127.0.0.1:8000/
